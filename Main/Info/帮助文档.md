# 帮助文档
- 1-项目已停止支持
    - Q1:为什么停止支持?
        A1:一些项目是限时的，因此会在到期时停止支持，没有更新计划的项目也会停止更新。
    - Q2:为什么有的项目仍提供下载链接，有的却无法再下载？
        A2:部分项目包含免费API或其它限时内容，当它们失效并导致程序无法运行时我们将会移除下载链接，大部分项目则能继续运行，并提供最后一个版本的下载链接。
    - Q3:停止更新的项目意味着什么？
        A3:虽然也属于停止了支持，但你还可能会收到安全更新，以保证正常使用。因为没有更新计划而导致停止支持的项目有一定概率重新启用，并继续更新。

- 2-为什么只提供最新版本的下载链接
    旧版本通常有漏洞或BUG在新版本中被修复，我们不希望您隐私泄露，也为支持更多功能，只提供最新版本下载。
