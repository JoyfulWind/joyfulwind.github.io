- Joyful Compress
    
    [Download](https://wwbqj.lanzout.com/iFpXT3n1gyyh)
    
    Version:1.0.0
    
    - Information
        
        这个项目能帮你压缩多个文件或文件夹，并且可以自定义字典大小和密码，采用AES 加密，几乎无法被暴力破解。低占用、温和后台压缩，游戏办公不卡顿，安静不打扰。
        
        注:这个项目的可维护性很低，不会像File Controller那样频繁地更新
        
        压缩和解压缩的格式均为.jcp文件，全名为joyful compress package
        
        警告:只能打包所有文件，无法打包文件夹，打包文件夹内的所有文件夹中的文件将会和打包文件夹下的其它文件放在一起，最好只用来打包文档、视频、音频，若要打包类似游戏等区分文件夹的文件需要先套一层压缩包再用Joyful Compress打包
