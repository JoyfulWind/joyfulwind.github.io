# File Controller
File Controller 使用指南
软件版本:1.0.9
更新时间:2026年3月6日20:37:51
第一步:输入你希望操作文件的路径(路径末尾带不带\都行)，并按下右边的确定按钮，此时文件浏览器会显示该文件夹下的所有文件(不包括文件夹)。每次只会展示5个文件，可以单击“向上查找”和“向下查找”来查找其余的文件。
第二步:输入你希望操作的文件名称(包括拓展名)，并点击确定来进行查找，若查找到将会显示”已找到文件”。
第三步:对文件进行相关的操作，接下来会介绍每个按钮的功能。
按钮功能介绍:
删除按钮:单击“删除”即可删除你选中的文件。
重命名按钮:单击“重命名”可以对文件进行重命名操作，此时控制中心将会自动显示重命名操作框，在框中输入重命名的名称后按下“确定”即可重命名成功。
打开按钮:单击”打开”可以打开选择的文件，需要有相应的默认软件来打开这个文件，若没有默认软件关联，则Windows会自动弹出”你要如何打开这个软件？”，选择相应软件即可打开。
移动按钮:单击“移动”可以对文件进行移动操作，此时控制中心会自动显示移动操作框，在框中输入要移动到的路径后按下“确定”即可移动成功。
注意事项:
1.重命名功能无法重命名为已存在文件的文件名
2.移动功能需要移动到存在的文件夹
3.需要有足够权限才能对文件进行相关操作
